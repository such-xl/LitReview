# 项目命名说明

## 项目名称：LitReview

### 全称
**LitReview** - Literature Review Assistant (文献综述助手)

### 命名理由

1. **简洁明了**
   - "Lit" = Literature（文献）
   - "Review" = 综述/评审
   - 组合起来直接表达核心功能

2. **易于记忆**
   - 短小精悍，只有9个字母
   - 发音简单，国际化友好
   - 符合学术软件命名习惯

3. **专业性强**
   - 直接关联学术研究领域
   - 体现系统的核心价值
   - 适合学术圈推广

4. **品牌潜力**
   - 域名友好（litreview.com等）
   - 易于搜索引擎优化
   - 便于社交媒体传播

### Logo建议
📚 或 📖 作为图标，配合蓝色/紫色学术风格配色

### Slogan建议
- "Smart Literature Review, Made Easy"
- "AI-Powered Research Assistant"
- "从论文到综述，一键完成"
